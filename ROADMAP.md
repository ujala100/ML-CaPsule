# 🗺️ ML-CaPsule Learning Roadmap

> A structured 3-tier learning path across 150+ projects.  
> Built for beginners who don't know where to start.

---

## How to Use This Roadmap

1. Start from 🟢 Beginner if you're new to ML
2. Move to 🟡 Intermediate once you're comfortable with basics
3. Tackle 🔴 Advanced when you're ready for complex models
